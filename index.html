Project 4: Voice-Controlled Device Management with Custom OAuth2 Authorization Server



Project 4 introduces a cutting-edge solution that seamlessly integrates Alexa Skill for voice-controlled device management. The project incorporates a custom OAuth2 authorization server, prioritizing security and ensuring a protected communication channel between Alexa and the user's devices.

The entire system is hosted on an AWS EC2 instance running Ubuntu OS, with the backend developed using the Django framework. This choice of technology not only facilitates rapid development but also ensures a robust and scalable solution. The deployment on AWS guarantees high availability, allowing users to access and control their devices at any time.

The user journey begins by enabling the Alexa Skill, triggering a secure login process. The custom OAuth2 authorization server handles the authentication, ensuring that only authorized users gain access. Post authentication, the system intelligently identifies the user's device, granting control over specific functionalities, such as managing lights.

A distinctive feature of this project is the emphasis on security. The custom OAuth2 server enhances the overall privacy and protection of user data during interactions with Alexa. This project serves as a testament to the fusion of voice technology, secure authentication mechanisms, and cloud-based hosting for efficient device control.
